# App Integrations

Pigeon's architecture is designed for seamless integration with DeFi protocols, apps, and platforms across the ecosystem. We continuously expand our toolkit to bring new capabilities directly into your chat interface.

## How Integrations Work

When we integrate a platform, you get:

- **Natural Language Access**: Interact with the protocol using plain English
- **Unified Context**: Combined with your portfolio, positions, and preferences
- **Pigeon Code Support**: Automate actions and monitoring via [Pigeon Code](pigeon-code.md)
- **Cross-Platform Synergy**: Combine with other tools for sophisticated strategies

## Featured Integrations

### Hypesino

[Hypesino](https://hypesino.com) is a decentralized casino built on Hyperliquid. Through Pigeon, you can:

- Play casino games directly from chat
- Track your gaming history and statistics
- Set responsible gaming limits
- Combine with Hyperliquid trading in unified workflows

```
"Play a round on Hypesino"
"What's my Hypesino balance?"
```

### Megapot

[Megapot](https://megapot.io) is a decentralized lottery jackpot on Base. Through Pigeon, you can:

- Purchase lottery tickets using USDC
- Check current jackpot size and odds
- Track your ticket history
- Get notified about drawing results

```
"Buy 5 Megapot tickets"
"What's the current Megapot jackpot?"
```
